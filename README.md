# Surfn Mint-X-Grey icon theme

A variant of the [Surfn](https://github.com/erikdubois/surfn) icon theme. Ships only the
**Surfn-Mint-X-Grey** theme; it inherits the base **Surfn** set, so `surfn-icons-git` is required.

## Installation

```
sudo pacman -S surfn-mint-x-grey-icons-git
```

`surfn-icons-git` (base Surfn) is pulled in automatically as a dependency.

## Manual

Copy `usr/share/icons/Surfn-Mint-X-Grey` into `~/.icons` (or `~/.local/share/icons` on Plasma),
ensure the base `Surfn` theme is present, then select **Surfn Mint-X-Grey** in your appearance settings.
